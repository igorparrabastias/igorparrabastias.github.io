# Algoritmos

Las recetas paso a paso: buscar, ordenar, recorrer grafos y encontrar caminos. Cada enlace abre el notebook renderizado en el sitio.

> [!TIP]
> Un algoritmo es una receta; la [estructura de datos](../estructuras/index.md) es la despensa. Sin buenos ingredientes ordenados, hasta la mejor receta tarda. ⏱️
