# Ordenamiento

Ordenar es reorganizar una colección bajo un criterio (normalmente de menor a mayor). Es la operación base de medio mundo: una vez que los datos están ordenados, buscar, deduplicar o agrupar se vuelve barato. Aquí están los tres clásicos —burbuja, mezcla y rápido— con sus ejercicios, más una guía para elegir cuál usar.

> [!TIP]
> No existe "el mejor" ordenamiento: existe el mejor **para tu caso**. Datos casi ordenados, memoria escasa o estabilidad requerida cambian por completo la respuesta. ⚖️
