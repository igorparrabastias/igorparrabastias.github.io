# Paradigmas de programación

Un mismo problema se puede pensar de formas muy distintas según el paradigma. Aquí el
**lenguaje es lo de menos**: lo interesante es la *forma de pensar* que cada uno impone.

> [!TIP]
> Aprender un paradigma nuevo no es aprender otra sintaxis: es que te crezca otra forma de
> resolver problemas. Luego vuelves a tu lenguaje de siempre y programas mejor. 🧠
