# Cómo funciona la computación

Una rampa de entrada **conceptual y sin requisitos** al resto de estos apuntes. Antes de escribir una línea de código conviene tener un mapa mental claro de qué es un computador, qué significa "computar" y por qué los lenguajes de programación existen.

Estas páginas no son técnicas: están pensadas para leerse de corrido, entender las ideas grandes y, desde ahí, saltar a las secciones más profundas. Si ya programas, sírvete de ellas como repaso o como material para explicarle a alguien que empieza.

> [!NOTE]
> El hilo conceptual sigue el enfoque del curso **MITx 6.00.1x** (*Introduction to Computer Science and Programming Using Python*), reescrito y resumido con ejemplos propios. Las fuentes se citan al final de cada página.
